En esta carpeta se almacenan todos los diseños que se mostrarán en el 
frontEnd, es decir, los nodos y relaciones.

NUEVA ESTRUCTURA CON REACT FLOW:

Esta carpeta ahora contiene la integración completa de React Flow para crear
diagramas interactivos de elementos del mundo. Incluye:

- FlowManager.js: Gestor principal del flujo y comunicación con backend
- FlowInterface.js: Interfaz de usuario y modales
- FlowInitializer.js: Inicializador de la aplicación
- FlowStyles.css: Estilos CSS para todos los componentes
- nodes/: Tipos de nodos personalizados (personajes, organizaciones, etc.)
- edges/: Tipos de conexiones personalizadas (relaciones, jerarquías, etc.)

Los nodos reciben datos directamente de la base de datos a través de las APIs
del backend y se pueden editar, crear y eliminar dinámicamente.

CARACTERÍSTICAS:
- Nodos personalizados para diferentes tipos de entidades
- Conexiones inteligentes entre elementos
- Interfaz moderna con drag & drop
- Integración completa con el backend
- Modo de simulación como fallback
- Exportación de diagramas
- Responsive design
