# SKILL: BACKEND SERVICE EXPERT (JAVA SPRING)

Estandarización de la capa lógica y de datos del servidor.

## ⚙️ Core Stack
- **Versión:** Java 21 LTS / Spring Boot 3.x.
- **Patrón:** DTO -> Controller -> Service -> Repository.

## 🔍 Reglas de Implementación
1. **Persistencia:** Siempre validar existencia de paths antes de escribir archivos JSON.
2. **Jerarquía:** No saltar niveles jerárquicos (Universo debe ser el padre de toda operación).
3. **Documentación Javadoc:** Obligatoria en métodos de servicio complejos.
