Acceso a Chronos Atlas
Esta es la primera interacción del usuario con la aplicación, actuando como la puerta de entrada. Permite a los nuevos usuarios Registrarse para crear una cuenta, mientras que los usuarios existentes pueden Iniciar Sesión utilizando sus credenciales. También ofrece una opción para Recuperar la Contraseña en caso de que el usuario la haya olvidado. Su diseño es limpio y se enfoca en la acción principal, minimizando distracciones.

Librería de Proyectos (Cuadernos)
Tras el acceso, esta pantalla sirve como el "hub" central para todos los proyectos del escritor. Aquí se muestra una lista organizada de los "Cuadernos" (proyectos) existentes. El usuario puede seleccionar un cuaderno para continuar trabajando en él, o crear un Nuevo Cuaderno para empezar un universo narrativo desde cero. Es el punto de partida para sumergirse en la creación de mundos.

Gestión de Hojas (Cuaderno Abierto)
Una vez que un cuaderno ha sido seleccionado, esta pantalla funciona como su "índice". Presenta una lista clara y ordenada de todas las "Hojas" que componen ese proyecto. Los usuarios pueden abrir una hoja específica para editarla, reordenar las hojas para organizar su narrativa, y crear nuevas hojas para expandir su historia. La navegación de "migas de pan" en la parte superior asegura que el usuario siempre sepa en qué cuaderno y contexto se encuentra.

Navegación Principal (Modos de App)
Esta es la herramienta central para cambiar entre las diferentes facetas de la aplicación. Ubicada en la parte superior central, se presenta como una discreta raya horizontal cuando está colapsada. Al pasar el ratón sobre ella, se despliega en un menú semicircular que permite al usuario seleccionar entre los modos principales de la aplicación: Escritura, Biblia del Mundo, Cartografía y Cronología. Al seleccionar un modo, la interfaz principal de la aplicación se transformará para mostrar las herramientas y vistas específicas de ese módulo, manteniendo el resto de la interfaz limpia y enfocada en la tarea actual.

Lienzo de Escritura (Editor)
Este es el corazón de la creación narrativa. Diseñado como un entorno libre de distracciones, permite al escritor concentrarse plenamente en su prosa. El texto se organiza en "hojas" numeradas con transiciones suaves, emulando la experiencia de un cuaderno físico. Su funcionalidad más innovadora es el Sistema de Referencias Inteligentes (@): al escribir @, se despliega un menú de autocompletado que sugiere entidades de la "Biblia del Mundo", permitiendo al escritor vincular su texto directamente con el lore sin salir del editor. Opcionalmente, se puede abrir un panel lateral colapsable para notas rápidas o ideas.

Creación Flotante de Entidad
Una funcionalidad directamente vinculada al Lienzo de Escritura, diseñada para mantener el flujo creativo. Cuando el usuario invoca una referencia inteligente @ y la entidad deseada no existe, o si decide crear una nueva, un modal flotante aparece sobre el texto. Este modal proporciona un formulario conciso para rellenar los campos esenciales de una nueva entidad (nombre, tipo, descripción breve), permitiendo al escritor crear y guardar la entidad rápidamente para que se integre en la Biblia del Mundo y se referencie automáticamente en el texto.

Biblia del Mundo (Gestión de Entidades)
Este módulo es la base de datos central de todo el universo narrativo del escritor. Permite la gestión organizada de entidades clasificadas en categorías predefinidas (Personajes, Lugares, Objetos, etc.). Los usuarios pueden visualizar listados tabulares, filtrar y ordenar las entidades por diversos criterios, y crear nuevas entidades. Es la fuente de verdad del mundo, asegurando coherencia y fácil acceso a toda la información del lore.

Ficha de Personaje (Entidad Detallada)
Como ejemplo de una entidad detallada en la Biblia del Mundo, esta pantalla permite ver y editar en profundidad la información de un personaje (o cualquier otra entidad). La información se presenta en secciones organizadas (Biografía, Atributos, Relaciones, etc.), utilizando campos personalizables según la plantilla definida. Ofrece opciones claras para Editar la ficha, Eliminar la entidad y, eventualmente, mostrará las referencias a esta entidad en el Lienzo de Escritura.

Gestión de Plantillas de Entidad
Complementando la Biblia del Mundo, esta pantalla permite una personalización profunda. Los usuarios pueden crear y gestionar plantillas para las fichas de sus entidades. Esto implica definir y reordenar campos de diversos tipos (texto, numérico, fecha, lista, etc.) para cada categoría de entidad. Es una herramienta potente para adaptar la estructura del lore a las necesidades específicas de cada historia.

Atlas Interactivo (Módulo Cartográfico)
Una herramienta geoespacial que va más allá de un simple mapa. Permite la visualización de mapas anidados, lo que significa que un usuario puede hacer zoom desde un mapa mundial a un continente, luego a una región, una ciudad, y finalmente el plano de un edificio. Los Pines Interactivos en el mapa están vinculados a entidades de la Biblia del Mundo, mostrando resúmenes rápidos al pasar el ratón y permitiendo acceso directo a las fichas detalladas con un clic.

Línea de Tiempo (Módulo Cronológico)
Esencial para cualquier narrativa, este módulo permite a los escritores crear y gestionar múltiples líneas de tiempo. Los usuarios pueden registrar eventos históricos, seguir arcos de personajes y visualizar secuencias de la trama de manera clara. Una función avanzada permitirá "deslizarse" por la línea de tiempo, actualizando el estado del mundo para reflejar una época específica, ofreciendo una fidelidad histórica sin precedentes.

Taller de Conlangs (Módulo Lingüístico)
Este módulo es una suite profesional para la creación de lenguajes construidos (conlangs). Incluye un Gestor de Léxico (diccionario), un Motor Gramatical para definir reglas fonológicas, morfológicas y sintácticas, y un Diseñador de Ortografía para crear sistemas de escritura personalizados. Además, se expande con: * Diseño Lingüístico: Para procesar texto, desglosarlo semánticamente y mapearlo a un léxico personalizado, permitiendo definir reglas complejas. * Editor Gráfico Versátil: Una herramienta avanzada que normaliza trazos dibujados a mano, imágenes o caracteres existentes en datos vectoriales escalables para la creación de glifos. * Traductor Logográfico en Tiempo Real: Renderiza visualmente la escritura logográfica creada a medida que el usuario escribe, ofreciendo una vista dinámica de la lengua inventada. * Generador de Fuentes: Compila los símbolos diseñados en un archivo de fuente tipográfica estándar e instalable, permitiendo utilizar la lengua inventada en cualquier entorno digital fuera de Chronos Atlas.

Perfil de Usuario
Esta pantalla permite al usuario ver y editar su información personal, incluyendo nombre de usuario y correo electrónico. También ofrece la opción de cambiar la contraseña, asegurando la seguridad de la cuenta. Todos los cambios se pueden guardar fácilmente.
