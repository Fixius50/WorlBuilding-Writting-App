Changelog / Features

Local-First

Latencia 0ms. La base de datos vive en tu navegador. Funciona sin internet.

Causalidad Profunda

No se guardan estados, se guarda historia. Puedes viajar a cualquier punto del tiempo.

Sincronización

De local a la nube (Supabase) automáticamente cuando hay conexión.

Visualización: Mapas vectoriales 8K y renderizado WebGL para tokens y ejércitos.

Navegación Radial

Menú superior "Top-Center" sin distracciones. Acceso rápido a módulos con un clic.

Multi-Cuenta

Gestión de usuarios y proyectos aislados. Seguridad integrada desde el login.
