Arquitectura (Capa API)

El frontend utiliza un objeto global API como wrapper para todas las interacciones.

// Estructura del Cliente API (api.js)
const API = {
    proyectos: {
        activo(), // Obtiene el proyecto actual
        listar(), // Lista todos los proyectos locales
        abrir(nombre)
    },
    bd: {
        listar(tipo), // Personajes, Lugares, etc.
        insertar(tipo, datos),
        modificar(tipo, id, datos)
    },
    escritura: {
        listarHojas(cuadernoId),
        guardarHoja(hojaId, contenido)
    }
}

Manual de Escritura Inmersiva

1

Referencias Inteligentes (@)

Escribe @ para desplegar el menú de entidades. Puedes seleccionar un personaje existente o crear uno nuevo sin abandonar el lienzo.

2

Modo Enfoque

Activa el Modo Enfoque desde el header para ocultar el sidebar de notas y centrarte exclusivamente en el texto. Todo el ruido visual desaparecerá.

3

Paginación Física

La escritura no es un infinito scroll. Cada hoja tiene un número asignado (@1, @2). Usa las flechas de navegación para transicionar entre ellas.
